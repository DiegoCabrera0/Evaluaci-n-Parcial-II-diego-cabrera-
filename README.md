- Proyecto en C# (.NET) `EscuelaAPI.sln`.
- Para ejecutar:
  1. Abrir `EscuelaAPI.sln` en Visual Studio.
  2. Ejecutar el proyecto (`F5` o `Ctrl+F5`).

## Frontend
- Proyecto Angular (`escuela-frontend`).
- Para ejecutar:
  1. Abrir cmd en `C:\Users\DELL\escuela-frontend`.
  2. Instalar dependencias: `npm install`.
  3. Ejecutar: `ng serve`.
  4. Abrir navegador en `http://localhost:4200`.
